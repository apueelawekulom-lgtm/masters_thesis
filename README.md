## File List

* **hvstat_africa_data_v1.0.csv**
  A CSV file containing harmonized subnational crop statistics.
* **hvstat_africa_boundary_v1.0.gpkg**\
  A GeoPackage file with FEWS NET’s administrative boundaries, aligned with crop statistics via FNID.
